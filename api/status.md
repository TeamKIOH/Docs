# API Status

## Status 코드는 아래와 같습니다.

| Status | Description            |
| ------ | ---------------------- |
| 100    | API에 접근이 실패하였습니다.      |
| 102    | 조회를 요청한 IP가 차단되었습니다.   |
| 103    | 조회를 요청한 지역이 차단되었습니다.   |
| 200    | 성공적으로 API를 호출했습니다.     |
| 202    | API호출을 정상적으로 하지 않았습니다. |
| 402    | API Limite             |
| 404    | 페이지를 찾을수 없습니다.         |
| 405    | API 에러로 인해 정상 호출실패     |
| 437    | API Key가 올바르지 않음       |
